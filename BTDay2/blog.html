Blog - Todo List Tutorial

1) Mục tiêu & demo chức năng

Ứng dụng Todo List sẽ có:

  • Ô nhập nội dung công việc
  • Nút Thêm
  • Danh sách công việc hiển thị bên dưới
  • Mỗi công việc có nút Xóa
  • (Thêm “xịn” một chút) Lưu dữ liệu vào localStorage để tải lại trang vẫn còn

2) Tạo cấu trúc dự án

Bạn tạo 3 file:

  • index.html
  • style.css
  • app.js

3) Bước 1 - Tạo giao diện HTML

Mô tả:
Ta dựng khung giao diện gồm tiêu đề, ô input, nút thêm và danh sách hiển thị. 


<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  <title>Todo List - HTML CSS JS</title>
  <link rel="stylesheet" href="style.css" />
</head>
<body>
  <div class="app">
    <h1 class="title">✅ Todo List</h1>

    <div class="todo-form">
      <input id="todoInput" type="text" placeholder="Nhập việc cần làm..." />
      <button id="addBtn">Thêm</button>
    </div>

    <ul id="todoList" class="todo-list"></ul>
  </div>

  <script src="app.js"></script>
</body>
</html>

Giải thích:

  • #todoInput là ô nhập công việc.
  • #addBtn là nút thêm.
  • #todoList là nơi hiển thị danh sách dưới dạng ul.
  • File JS đặt cuối body để đảm bảo HTML tải xong rồi JS mới chạy.

Bước 3 — JavaScript: thêm, hiển thị, xóa (có lưu localStorage)

Mô tả:
Ta sẽ:

  1. Lấy các element từ HTML
  2. Tạo mảng todos
  3. Viết hàm render() để vẽ danh sách
  4. Viết hàm addTodo() để thêm
  5. Viết hàm deleteTodo() để xóa
  6. Lưu/đọc dữ liệu từ localStorage
// app.js

// 1) Lấy phần tử HTML
const todoInput = document.getElementById("todoInput");
const addBtn = document.getElementById("addBtn");
const todoList = document.getElementById("todoList");

// 2) Load dữ liệu từ localStorage (nếu có)
const STORAGE_KEY = "todos_v1";
let todos = JSON.parse(localStorage.getItem(STORAGE_KEY)) || [];

// 3) Hàm lưu dữ liệu
function saveTodos() {
  localStorage.setItem(STORAGE_KEY, JSON.stringify(todos));
}

// 4) Hàm hiển thị danh sách
function render() {
  // Xóa UI cũ
  todoList.innerHTML = "";

  // Nếu rỗng thì hiện trạng thái empty
  if (todos.length === 0) {
    const li = document.createElement("li");
    li.className = "empty";
    li.textContent = "Chưa có công việc nào. Hãy thêm việc mới!";
    todoList.appendChild(li);
    return;
  }

  // Tạo UI cho từng todo
  todos.forEach((todo) => {
    const li = document.createElement("li");
    li.className = "todo-item";

    const span = document.createElement("span");
    span.className = "todo-text";
    span.textContent = todo.text;

    const delBtn = document.createElement("button");
    delBtn.className = "delete-btn";
    delBtn.textContent = "Xóa";

    // Khi bấm xóa
    delBtn.addEventListener("click", () => {
      deleteTodo(todo.id);
    });

    li.appendChild(span);
    li.appendChild(delBtn);
    todoList.appendChild(li);
  });
}

// 5) Hàm thêm todo
function addTodo() {
  const text = todoInput.value.trim();

  // Chặn thêm rỗng
  if (!text) return;

  const newTodo = {
    id: Date.now(), // tạo id đơn giản bằng timestamp
    text: text,
  };

  todos.unshift(newTodo); // thêm lên đầu danh sách
  saveTodos();
  render();

  // reset input
  todoInput.value = "";
  todoInput.focus();
}

// 6) Hàm xóa todo theo id
function deleteTodo(id) {
  todos = todos.filter((t) => t.id !== id);
  saveTodos();
  render();
}

// 7) Gắn sự kiện
addBtn.addEventListener("click", addTodo);

// Enter để thêm nhanh
todoInput.addEventListener("keydown", (e) => {
  if (e.key === "Enter") addTodo();
});

// 8) Render lần đầu
render();

Giải thích:

  • todos là mảng dữ liệu chính, mỗi phần tử có {id, text}.
  • render() chịu trách nhiệm hiển thị: luôn xóa UI cũ rồi vẽ lại từ dữ liệu mới.
  • addTodo():
    • lấy text từ input, trim() để bỏ khoảng trắng
    • nếu rỗng thì không thêm
    • tạo id bằng Date.now()
    • unshift() để todo mới nằm trên cùng
  • deleteTodo(id) dùng filter() để loại todo có id tương ứng.
  • localStorage giúp lưu lại dữ liệu khi refresh trang.

Kiểm tra nhanh

Bạn mở index.html lên và thử:

  • Nhập “Học JavaScript” → Thêm
  • Nhấn Enter để thêm nhanh
  • Nhấn Xóa để xóa 1 dòng
  • Refresh trang → dữ liệu vẫn còn
